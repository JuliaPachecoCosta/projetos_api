## 💻 Sobre o projetos_api

O **projetos_api** contém uma série de exercícios práticos focados no aprendizado de JavaScript Vanilla (puro) para integração com APIs externas e manipulação de dados.

Este projeto foi desenvolvido como parte do currículo do curso **Técnico em Desenvolvimento de Sistemas** da Escola **SENAI A. Jacob Lafer**.

---

## ⚙️ Funcionalidades e Conceitos Aplicados

Durante o desenvolvimento dos exercícios, foram aplicados os seguintes conceitos:

* **Manipulação do DOM:** Seleção e alteração dinâmica de elementos HTML.
* **Requisições Assíncronas:** Consumo de APIs utilizando `fetch()`.
* **Async/Await:** Escrita de código assíncrono moderno e organizado.
* **Tratamento de Dados JSON:** Filtragem e mapeamento de respostas de APIs.
* **Tratamento de Erros:** Uso de `try...catch` para lidar com falhas de conexão.
* **Interface Dinâmica:** Criação de elementos de UI (cards, listas, tabelas) via JS.
